# Documentação GamerVerse

## Especificação de UX, Dados e API

- **`GamerVerse-UX-e-API-Spec.md`** – Versão em Markdown (fácil de editar e versionar).
- **`GamerVerse-UX-e-API-Spec.html`** – Versão para abrir no **Microsoft Word**.

### Como abrir no Word

1. Abra o **Microsoft Word**.
2. Vá em **Arquivo → Abrir** e selecione o arquivo **`GamerVerse-UX-e-API-Spec.html`**.
3. O Word abrirá o documento com formatação e tabelas.
4. Para salvar como `.docx`: **Arquivo → Salvar como** e escolha o formato **Documento do Word (.docx)**.

Alternativa: abra o `.html` no navegador, copie o conteúdo e cole no Word (a formatação das tabelas pode ser preservada).
